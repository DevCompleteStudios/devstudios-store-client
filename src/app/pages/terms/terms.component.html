Información del usuario y seguridad

Nosotros no guardamos datos del cliente como tarjetas de credito, etc. Lo que guardamos solamente son los datos necesarios para iniciar sesión, crear cuentas y acceder a tus compras y datos como transacciónes de pagos. Tus datos estan seguros y protegidos ya que nadie tiene acceso a ellos a menos de que se te pida para arreglar algún error con tu compra.

Error en la compra

Si tu compra tuvo algun incoveniente, sera manejado lo antes posible para darte una solución. En caso de que el pago haya sido realizado y el producto o servicio no este funcionando, deberas esperar máximo 3 días para poder tener tu compra, si no fue así tendras la opción de tener reembolso o una compensación extra.
